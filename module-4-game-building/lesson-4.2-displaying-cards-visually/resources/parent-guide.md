| Challenge | Why It Happens | How to Help |
|-----------|----------------|-------------|
| Cards look stretched | Wrong aspect ratio | Check width/height proportions (5:7) |
| Animations are choppy | Not using transitions | Add transition property to smoothly animate |
| Layout breaks on mobile | Not responsive | Test with browser dev tools, add media queries |
| Colors look wrong | Incorrect color codes | Verify red (#d32f2f) and black (#333) |

## üîß Troubleshooting Tips

If your child is stuck:
1. **Check the browser console** for errors (F12)
2. **Use browser dev tools** to inspect elements
3. **Compare with examples** to spot differences
4. **Start simple** then add complexity gradually
5. **Test frequently** to catch issues early
