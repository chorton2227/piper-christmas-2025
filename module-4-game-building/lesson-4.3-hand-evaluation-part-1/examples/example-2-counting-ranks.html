The Frequency Map

To detect pairs, three of a kind, and four of a kind, we need to count how many cards we have of each rank.

function countRanks(cards) { const counts = {}; for (const card of cards) { const rank = card.rank; counts[rank] = (counts[rank] || 0) + 1; } return counts; }

How it works:

  • Create an empty object to store counts
  • Loop through each card
  • If the rank doesn't exist in counts, set it to 0, then add 1
  • Return the counts object
