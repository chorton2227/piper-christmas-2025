# Exercise 3: Set Up Your GitHub Profile (Optional)

## Objective

Personalize your GitHub profile to make it professional and presentable.

## Instructions

### Step 1: Navigate to Your Profile

1. Log in to GitHub
2. Click your profile icon in the top-right
3. Click "Your profile"

You'll see your profile page with your username and join date.

### Step 2: Edit Your Profile

Click the **"Edit profile"** button.

### Step 3: Add Profile Information

Complete these optional fields:

#### Profile Picture
- Click on your current avatar
- Upload a photo or avatar image
- **Tips:**
  - Use a clear image
  - Professional or fun (your choice)
  - Square images work best
  - Many students use avatars/logos they created

#### Bio
Write a short bio (160 characters max)

**Examples:**
- "Student developer learning web development üöÄ"
- "High school coder | Building games with HTML, CSS, and JavaScript"
- "Learning to code one project at a time üíª"
- "Web development student | Texas Hold'em poker game creator"

#### Company
- Optional field
- You could put your school name or leave blank
- Example: "Lincoln High School" or "Student"

#### Location
- Optional
- Can be as general as you want
- Examples: "USA", "California", "Sacramento, CA"

#### Website
- Leave blank for now

#### Twitter
- Optional
- Only if you have a Twitter account and want to link it

### Step 4: Customize Your Profile Appearance

Scroll down to see more options:

#### Profile README (Advanced - Optional)
You can create a special repository that displays on your profile.

**For now:** Skip this - you can add it later after you've learned more!

#### Pinned Repositories
After you create your poker game repository, you can "pin" it to your profile to highlight it.

**For now:** Nothing to pin yet - we'll come back to this!

### Step 5: Save Your Changes

Click **"Save"** at the bottom of the page.

### Step 6: View Your Public Profile

Click "Overview" to see how your profile looks to others.
