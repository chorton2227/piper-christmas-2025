## What is GitHub?

**GitHub is like Google Drive for code.** It's a website where developers:
- Store their code
- Share projects with others
- Host websites for free
- Work on code together

### Why GitHub?

- **Free hosting** - GitHub Pages lets you host websites at no cost
- **Professional tool** - Used by millions of developers worldwide
- **Portfolio** - Show off your projects to colleges/employers
- **Backup** - Your code is safely stored in the cloud

### Real-World Use

- Companies like Google, Microsoft, and Facebook use GitHub
- Open-source projects (like VS Code!) are hosted on GitHub
- Developers share code and collaborate globally

## Key GitHub Terms

Before we dive in, let's learn some important vocabulary:

### Repository (or "repo")
A **repository** is like a folder for your project. It contains all your files (HTML, CSS, JavaScript) and tracks changes to them.

**Example:** `poker-game` repository contains your Texas Hold'em game

### Commit
A **commit** is like saving a version of your project. Each commit is a snapshot of your files at a specific time.

**Example:** "Added betting feature" commit shows what changed when you added betting

### GitHub Pages
**GitHub Pages** is free website hosting from GitHub. It turns your repository into a live website.

**Example:** Your repo becomes `https://yourusername.github.io/poker-game`

**Real example:** [https://chorton2227.github.io/poker-game/](https://chorton2227.github.io/poker-game/)

### Clone/Upload
- **Clone** = Download a repository from GitHub to your computer
- **Upload/Push** = Send your files from computer to GitHub

## Creating Your GitHub Account

Let's set up your free GitHub account!

### Step 1: Go to GitHub

Open your web browser and go to: **https://github.com**

### Step 2: Click "Sign Up"

Look for the "Sign up" button in the top-right corner.

### Step 3: Choose a Username

**Important:** Your username will be part of your website URL!

**Good usernames:**
- `piper-dev` ‚Üí `https://piper-dev.github.io/poker-game`
- `pipercodes` ‚Üí `https://pipercodes.github.io/poker-game`
- `piper2025` ‚Üí `https://piper2025.github.io/poker-game`
- `chorton2227` ‚Üí `https://chorton2227.github.io/poker-game` ‚Üê Real working example!

**Tips:**
- Keep it professional (you might use this for job applications!)
- Make it memorable
- Avoid numbers if possible (unless birth year)
- Can't change it easily later, so choose wisely

### Step 4: Enter Email and Password

- Use an email you can access (you'll need to verify it)
- Choose a strong password
- Complete the verification puzzle

### Step 5: Verify Your Email

1. GitHub will send you an email
2. Open it and click the verification link
3. You'll be redirected back to GitHub

### Step 6: Choose Free Plan

GitHub will ask about your plan:
- Select **"Free"** (you don't need paid features)
- Answer optional questions or skip them

**Congratulations! You now have a GitHub account!** üéâ

## Need Help?

**Common Issues:**

**Q: I didn't get the verification email**
- Check your spam folder
- Click "Resend verification email" on GitHub
- Make sure you entered the correct email address

**Q: My username is taken**
- Try adding numbers (birth year)
- Try a variation (add "dev" or "codes")
- Use middle name or nickname

**Q: I want to change my username later**
- It's possible but complicates things
- Better to choose carefully now
- URLs with old username won't work
