# Student Workspace - Lesson 5.1

Welcome to Module 5! You're about to learn how to host your Texas Hold'em poker game on the internet using GitHub.

## What You Need

Before starting this lesson, make sure you have:

- [ ] Web browser (Chrome, Firefox, Safari, or Edge)
- [ ] Email address you can access
- [ ] About 1 hour of time
- [ ] Your completed poker game from Module 4

## What You'll Do

In this lesson, you will:

1. **Learn about GitHub** - What it is and why developers use it
2. **Create a GitHub account** - Set up your free account
3. **Explore GitHub** - See how other developers organize projects
4. **Plan your repository** - Decide how to organize your poker game

## Important Decisions

You'll need to make these choices during this lesson:

### 1. GitHub Username
Choose carefully - this will be part of your website URL!

**Your username:**  
`_______________________________`

**Your future URL will be:**  
`https://[username].github.io/poker-game`

### 2. Repository Name
What will you call your poker game repository?

**Your repository name:**  
`_______________________________`

**Good examples:** `poker-game`, `texas-holdem`, `holdem-poker`

### 3. Project Description
One sentence describing your game:

## Exercises for This Lesson

### Exercise 1: Explore GitHub
- Browse existing projects on GitHub
- See how professionals organize code
- Get ideas for your own repository

**Time:** 15-20 minutes

### Exercise 2: Plan Your Repository
- Choose your repository name
- Write your project description
- Plan your file organization

**Time:** 15-20 minutes

### Exercise 3: Set Up Your Profile (Optional)
- Add a profile picture
- Write a bio
- Make your profile look professional

**Time:** 10-15 minutes
