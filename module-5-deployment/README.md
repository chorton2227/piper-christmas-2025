# Module 5: Host Your Poker Game on GitHub

## Overview

You've built an awesome Texas Hold'em poker game - now let's put it on the internet! In this module, you'll learn how to use GitHub Pages to host your game for free, giving it a real web address that anyone can visit.

**By the end:** Your poker game will be live at `https://yourusername.github.io/poker-game`

**Real example:** [https://chorton2227.github.io/poker-game/](https://chorton2227.github.io/poker-game/)

## What You'll Learn

- What GitHub is and why developers use it
- How to create a GitHub account
- Uploading your poker game to GitHub
- Deploying with GitHub Pages (free hosting!)
- Sharing your game with others

## Module Structure

### Lesson 5.1: Introduction to GitHub
**Time:** 30-45 minutes

- What is GitHub?
- Why use GitHub to host your game?
- Creating your free GitHub account
- Understanding GitHub's interface

**Goal:** Set up your GitHub account and understand how it works.

### Lesson 5.2: Upload Your Game
**Time:** 45-60 minutes

- Create a new repository
- Upload your HTML, CSS, and JavaScript files
- Write a README for your project
- Make your first commit

**Goal:** Your poker game files are on GitHub.

### Lesson 5.3: Deploy with GitHub Pages
**Time:** 45-60 minutes

- Enable GitHub Pages
- Get your live URL
- Test your deployed game
- Fix any issues
- Share with friends!

**Goal:** Your game is LIVE on the internet!

## Parent Note

This module teaches students to use **GitHub**, a professional developer platform.

- Upload code to a public repository (visible to anyone)
- Make their game accessible via public URL

**Please discuss:**
- Choosing an appropriate GitHub username
- Understanding that uploaded code is publicly visible
- Appropriate online presence and digital footprint

GitHub is a legitimate, professional platform used by developers worldwide and is appropriate for students learning to code.
