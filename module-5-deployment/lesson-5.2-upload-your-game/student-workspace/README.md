## Tasks for This Lesson

### Task 1: Create Repository
- Create your repository on GitHub
- Add repository name and description

### Task 2: Upload Files
- Upload index.html
- Create folder structure
- Upload CSS and JavaScript files
- Organize into folders

### Task 3: Update Paths
- Edit index.html to match new folder structure
- Update all `<link>` and `<script>` tags

### Task 4: Write README
- Create README.md
- Add title, description, features
- Explain how to play
- List technologies used

### Task 5: Verify
- Check folder structure
- Test file paths
- Review commit messages
- Ensure everything looks professional

## Practice Exercises

### Exercise 1: Upload and Organize (45 min)

### Exercise 2: Write README (30 min)

## Important Reminders

⚠️ **Main HTML file MUST be named `index.html`**

⚠️ **File paths in HTML must match folder structure**

⚠️ **Use clear commit messages for each change**

⚠️ **README.md must be in root directory**
