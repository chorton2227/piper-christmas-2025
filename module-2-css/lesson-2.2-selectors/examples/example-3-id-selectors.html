Understanding ID Selectors

ID selectors are used for unique elements on the page. Each ID can only be used ONCE per page.

IDs are great for:

  • Page sections (header, navigation, footer)
  • Unique components (main content area, sidebar)
  • Elements you want to target with JavaScript

⚠️ Important!

Remember: IDs must be unique! You can't have two elements with id="header" on the same page.

For styling, classes are usually better because they're reusable!

ID vs Class

Think of it this way:

  • IDs are like your Social Security Number - unique to you!
  • Classes are like your student group - multiple people can be in it!
