### 1. Selectors & Specificity
- Element selectors (body, h1, p, etc.)
- Class selectors (.header, .nav-link, .btn, etc.)
- Compound selectors (.nav-link:hover, .stat-card:hover)

### 2. Colors & Backgrounds
- Hex colors (#667eea, #764ba2, #2c3e50)
- RGBA for transparency (rgba(255,255,255,0.2))
- Linear gradients (linear-gradient(135deg, #667eea 0%, #764ba2 100%))

### 3. Typography
- Font families (Segoe UI, sans-serif)
- Font sizes (16px to 56px)
- Font weights (normal, bold, 500, 600)
- Line height for readability
- Text shadows

### 4. Box Model
- Margins for spacing between elements
- Padding for internal spacing
- Borders with colors and radius
- Width and height properties
- box-sizing: border-box

### 5. Borders & Shadows
- border-radius for rounded corners (4px, 8px, 12px)
- box-shadow for depth (0 4px 15px rgba(0,0,0,0.1))
- border colors and styles
- Transition effects on borders

### 6. Flexbox Layout
- Header navigation (display: flex, justify-content, gap)
- Hero centering (align-items, justify-content)
- Featured game layout (flex: 1, flex: 2)
- Platform cards (flex-wrap)
- Footer sections

### 7. CSS Grid
- Stats grid (grid-template-columns: repeat(auto-fit, minmax()))
- Responsive grid layout
- Gap spacing

### 8. Positioning
- Fixed header (position: fixed, z-index)
- Relative positioning for context
- Proper stacking context

### 9. Transitions & Hover Effects
- Button hover effects (transform, box-shadow)
- Navigation link hover (background, translateY)
- Card hover animations (translateY, scale)
- Smooth transitions (transition: all 0.3s ease)

### 10. Responsive Design
- Mobile-first approach
- Media queries (@media (max-width: 768px))
- Flexible layouts (flex-wrap, grid)
- Viewport meta tag
- Responsive typography
