Box-Sizing: content-box vs border-box

Default: content-box ‚ùå

Width applies to CONTENT ONLY. Padding and border are ADDED to the width.

CSS:

width: 300px;
padding: 20px;
border: 5px solid;

Actual Total Width:

300px (content) + 40px (padding) + 10px (border) = 350px

Better: border-box ‚úÖ

Width applies to the ENTIRE BOX. Padding and border are INCLUDED in the width.

CSS:

box-sizing: border-box;
width: 300px;
padding: 20px;
border: 5px solid;

Actual Total Width:

300px (includes everything!)

üåü Best Practice: Apply to Everything!

* {
  box-sizing: border-box;
}

This makes ALL elements use border-box, making sizing predictable and easier to work with!
