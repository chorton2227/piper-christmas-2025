Box Model Practice

Box 1: Only Padding

This box has 20px padding on all sides. The padding creates space between this text and the edges of the box. Notice how the background color extends into the padding area!

Box 2: Padding and Border

This box has 25px padding AND a 3px solid border. The border wraps around the padding, creating a visible boundary. The padding keeps the content from touching the border.

Box 3: Padding, Border, and Margin

This box has padding, border, AND 30px margin. The margin creates transparent space outside the border, pushing this box away from other elements. Notice the extra space around this box!

Box 4: Different Padding on Each Side

This box demonstrates different padding values on each side: Top: 10px, Right: 20px, Bottom: 30px, Left: 40px. Notice how the text is positioned differently from each edge!

Bonus: Centered Card

This card is centered using margin: 0 auto, has rounded corners with border-radius, and includes a subtle box-shadow for depth!
