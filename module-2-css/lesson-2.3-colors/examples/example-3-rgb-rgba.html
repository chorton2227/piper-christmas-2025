Understanding RGB and RGBA Colors

RGB uses three values (Red, Green, Blue) from 0-255. RGBA adds transparency with a fourth value (Alpha) from 0-1!

RGB Basics

rgb(255, 0, 0)

Pure Red

Red: 255 (max), Green: 0, Blue: 0

rgb(0, 255, 0)

Pure Green

Red: 0, Green: 255 (max), Blue: 0

rgb(0, 0, 255)

Pure Blue

Red: 0, Green: 0, Blue: 255 (max)

Mixed RGB Colors

rgb(128, 0, 128)

Purple

Red + Blue = Purple

rgb(255, 165, 0)

Orange

Lots of Red + Some Green

rgb(128, 128, 128)

Gray

Equal amounts = Gray
