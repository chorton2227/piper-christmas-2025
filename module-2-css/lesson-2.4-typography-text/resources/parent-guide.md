# Parent Guide: Typography & Text

## Overview
Lesson 2.4 introduces typography and text styling in CSS. Students will learn to control fonts, sizes, spacing, and formatting to create professional, readable text layouts.

## What Your Child is Learning

### Core Concepts
1. **Font Properties**
   - font-family (choosing typefaces)
   - font-size (controlling text size)
   - font-weight (boldness)
   - font-style (italic/normal)

2. **Text Properties**
   - text-align (alignment)
   - line-height (spacing between lines)
   - letter-spacing (space between characters)
   - text-decoration (underlines, strikethrough)
   - text-transform (uppercase, lowercase)

3. **Google Fonts**
   - Using free web fonts
   - Font pairing (serif + sans-serif)
   - Creating visual hierarchy

### Why This Matters
Typography is fundamental to web design. Good typography:
- Makes content easy and comfortable to read
- Creates visual hierarchy (showing what's important)
- Establishes mood and personality
- Improves user experience
- Makes websites look professional

These skills apply to any text-based design work!

## Lesson Structure

### Part 1: Font Properties (30 minutes)
Students explore different fonts and learn to apply them using CSS.

**What to expect:**
- Trying different font families

Students learn to style and format text for readability and visual appeal.

**What to expect:**
- Aligning text (center, left, right, justify)
- Adjusting line spacing for readability
- Adding decorations (underlines, strikethrough)
- Transforming text case

Students discover how to use beautiful free fonts from Google Fonts.

**What to expect:**
- Browsing and selecting fonts
- Adding fonts to HTML
- Pairing fonts effectively
- Creating professional typography

## Common Challenges & Solutions

### Challenge 1: "My font isn't changing!"
**Possible causes:**
- Font name is misspelled
- Forgot quotes around font names with spaces
- CSS file not linked properly
- Selector not matching the HTML element

**How to help:**
- Check spelling carefully: 'Times New Roman' not 'Times New Roman'
- Verify the CSS file is linked in HTML
- Use browser developer tools to inspect

---

### Challenge 2: "The text looks cramped/hard to read"
**Solution:**
- Adjust line-height (try 1.6 for body text)
- Increase font-size (minimum 16px for body)
- Add margin/padding around text
- Check color contrast

---

### Challenge 3: "Google Font isn't working"
**Possible causes:**
- Forgot to add the `<link>` in HTML head
- Font name misspelled in CSS
- Internet connection issue

**How to help:**
- Check the `<link>` tag is in `<head>` section
- Copy font name exactly from Google Fonts
- Verify font-family in CSS matches exactly (including quotes)

---

### Challenge 4: "Too many different fonts looks messy"
**Teaching moment:**
- Explain the 2-3 font maximum rule
- Show examples of good vs bad font pairing
- Discuss visual hierarchy vs chaos
- Practice restraint and consistency

## Vocabulary

- **Font Family:** The typeface used for text (Arial, Georgia, etc.)
- **Serif:** Fonts with decorative strokes (Times New Roman, Georgia)
- **Sans-Serif:** Clean fonts without strokes (Arial, Helvetica)
- **Monospace:** Fixed-width fonts (Courier, often used for code)
- **Font Stack:** List of fonts with fallbacks
- **Web-Safe Fonts:** Fonts available on all computers
- **Google Fonts:** Free custom fonts from Google
- **Font Weight:** How bold text is (100-900)
- **Line Height:** Vertical spacing between lines of text
- **Visual Hierarchy:** Using size/weight to show importance
- **Typography:** The art of arranging text

## Tips for Success

✅ **Encourage experimentation** - Try different fonts and sizes freely

✅ **Emphasize readability** - Good typography is comfortable to read

✅ **Practice restraint** - Less is more with fonts (2-3 maximum)

✅ **Study real examples** - Analyze typography on favorite websites

✅ **Test at different sizes** - Typography should work at various sizes

✅ **Consider the audience** - Fonts should match the content's purpose
