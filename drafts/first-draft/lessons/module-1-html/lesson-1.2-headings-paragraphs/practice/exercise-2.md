## ğŸ“‹ Instructions

1. Create a new file called `recipe.html`
2. Choose your favorite recipe (or make one up!)
3. Format it using HTML with:
   - `<h1>` for the recipe name
   - `<h2>` for "Ingredients" section
   - `<h2>` for "Instructions" section
   - `<strong>` for important words
   - `<em>` for emphasis
   - `<br>` for line breaks in the ingredient list
   - Other formatting as needed

### Example Structure:
```
H1: Chocolate Chip Cookies
  H2: Description
    Paragraph about the recipe
  H2: Ingredients
    Paragraph with ingredients (use <br> between each)
  H2: Instructions
    Multiple paragraphs, one for each step
  H2: Tips
    Helpful hints
```
