## ğŸ’¡ Tips for Success

### Starting Tips
- Don't try to do everything at once
- Build one page at a time
- Test frequently in your browser
- Save often (Ctrl+S)

### HTML Tips
- Use proper indentation (2 or 4 spaces)
- Close all your tags
- Use semantic elements when appropriate
- Add comments to explain complex sections

### Content Tips
- Write real content, not just "Lorem ipsum"
- Make sure text is appropriate and school-friendly
- Proofread for spelling and grammar
- Keep paragraphs to a reasonable length

### Image Tips
- Create an `images` folder
- Name files without spaces (use hyphens: `my-image.jpg`)
- Always add alt text
- Don't use huge image files

### Common Issues

**Links not working?**
- Check that filenames match exactly (including `.html`)
- Make sure the file exists in the same folder
- Use relative paths (just the filename, not the full path)

**Images not showing?**
- Check that the image file exists in the `images` folder
- Verify the filename matches exactly (including extension)
- Make sure path is correct: `images/filename.jpg`

**Form not doing anything?**
- That's normal! Forms need backend code to process
- For this project, the form just needs to exist and have proper structure

**Page looks messy?**
- HTML doesn't include stylingâ€”that's what CSS is for (Module 2!)
- Focus on structure and content for now
