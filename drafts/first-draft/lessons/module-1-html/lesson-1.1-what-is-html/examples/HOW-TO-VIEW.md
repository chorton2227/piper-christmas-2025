# How to View These Examples

## Method 1: Double-Click (Easiest!)
1. Find the `.html` file in your folder
2. Double-click it
3. It will open in your default web browser
4. That's it!

## Method 2: Right-Click and Open With
1. Right-click the `.html` file
2. Choose "Open with"
3. Select your favorite browser (Chrome, Firefox, Edge, etc.)
4. The page will open!

## Method 3: Drag and Drop
1. Open your web browser
2. Drag the `.html` file into the browser window
3. Drop it!
4. The page appears!

## Method 4: From VS Code
1. Right-click the `.html` file in VS Code
2. Look for "Open with Live Server" or "Open in Browser"
3. The page opens automatically!

---

## What You Should See

Each example shows you different HTML elements:

- **example-1**: Just one big heading
- **example-2**: All six heading sizes (h1 through h6)
- **example-3**: Headings and paragraphs together

---

## Try Changing Things!

Once you open an example:
1. Go back to VS Code
2. Change some text
3. Save the file (Ctrl+S or Cmd+S)
4. Refresh your browser (F5 or Ctrl+R)
5. See your changes!
