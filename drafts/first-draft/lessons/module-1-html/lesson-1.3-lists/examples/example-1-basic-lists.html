Understanding Lists

Unordered List (Bullet Points)

Use when order doesn't matter:

  • Apples
  • Bananas
  • Oranges
  • Grapes

Ordered List (Numbered)

Use when order is important:

  1. First step
  2. Second step
  3. Third step
  4. Fourth step

Comparison

Notice how unordered lists have bullets and ordered lists have numbers!
