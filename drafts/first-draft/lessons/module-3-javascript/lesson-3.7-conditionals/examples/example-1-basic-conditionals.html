Learning if, else, and else if statements

Demo 1: Simple if Statement

Enter your age to see if you're old enough to play the game!

if (age >= 10) { // Code runs if condition is true console.log("You're old enough!"); }
How it works: The code checks if your age is at least 10. If the condition is TRUE, the message appears. If it's FALSE, nothing happens.

Demo 2: if/else Statement

Check if it's warm or cool outside based on temperature!

if (temperature > 70) { console.log("It's warm outside!"); } else { console.log("It's cool outside!"); }
How it works: This uses BOTH if and else. If temperature > 70 is TRUE, it shows "warm". If FALSE, it shows "cool". One or the other ALWAYS runs.

Demo 3: else if Chain

Convert a test score to a letter grade!

if (score >= 90) { grade = "A"; } else if (score >= 80) { grade = "B"; } else if (score >= 70) { grade = "C"; } else if (score >= 60) { grade = "D"; } else { grade = "F"; }
How it works: JavaScript checks conditions from TOP TO BOTTOM. As soon as one is TRUE, it runs that block and STOPS checking the rest.

Demo 4: Multiple Independent if Statements

Enter a number and see which conditions are true!

if (num > 0) { console.log("Positive"); } if (num % 2 === 0) { console.log("Even"); } if (num >= 10) { console.log("Two digits or more"); }
How it works: These are SEPARATE if statements (not else if), so ALL of them are checked. Multiple conditions can be true at the same time!

Demo 5: Password Validator

Create a password and check if it's strong enough!

if (password.length < 6) { return "Too short!"; } else if (password.length < 8) { return "Okay, but could be longer"; } else if (password.length < 12) { return "Good length!"; } else { return "Excellent - very secure!"; }
How it works: This checks password length in stages. Shorter passwords get warnings, longer passwords get praise!

Demo 6: Number Comparison

Compare two numbers and see which is larger!

if (num1 > num2) { result = num1 + " is greater"; } else if (num1 < num2) { result = num2 + " is greater"; } else { result = "They are equal!"; }
How it works: This handles ALL THREE possibilities: first is bigger, second is bigger, or they're equal.
