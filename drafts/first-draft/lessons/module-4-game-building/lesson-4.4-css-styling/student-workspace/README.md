## üí° What I Learned

### Key Concepts

**CSS Syntax:**
```css
selector {
    property: value;
}
```

**Box Model:**
- Margin = Space outside
- Border = Edge line
- Padding = Space inside
- Content = The element itself

**Flexbox:**
- `display: flex;` = Use Flexbox
- `justify-content` = Horizontal alignment
- `align-items` = Vertical alignment

**Transitions vs Animations:**
- Transitions = Smooth change between two states (hover)
- Animations = Multi-step sequences with @keyframes

**Responsive Design:**
- @media queries change styles based on screen size
- Mobile-first approach is best practice
