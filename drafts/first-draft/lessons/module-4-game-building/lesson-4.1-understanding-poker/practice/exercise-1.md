**Goal**: Create physical flash cards to memorize poker hands

## üéÆ How to Use Your Flash Cards

### Game 1: Memory Test
1. Shuffle all cards
2. Look at the FRONT
3. Try to describe the hand WITHOUT looking at back
4. Check your answer
5. Keep practicing until you get all 10 correct!

### Game 2: Speed Challenge
1. Set a timer for 60 seconds
2. Try to name all 10 hands in order from worst to best
3. Beat your time!

### Game 3: Quiz a Parent
1. Show your parent the front of a card
2. They try to explain it
3. You check if they're right
4. Switch roles!

### Game 4: Draw Challenge
1. Parent/friend says a hand name
2. You draw an example WITHOUT looking
3. Check if you got it right

## üèÜ Bonus Challenge

**Make a "mini deck" of just 20 cards**:
- Pick 4 suits (or use colored paper)
- Make ranks: A, 2, 3, 4, 5
- Use these to practice making hands
- Try to make each type of hand!

**Example**:
- Can you make a flush? (all same color)
- Can you make a straight? (A-2-3-4-5)
- Can you make three of a kind? (three 3s)
