flex-shrink: Handling Tight Spaces

Controls how items shrink when space is limited (default is 1)

flex-shrink: 0 (won't shrink)
flex-shrink: 1 (shrinks)
/* Item will shrink if needed (default) */ .item { flex-shrink: 1; } /* Item won't shrink, keeps its size */ .item { flex-shrink: 0; }
