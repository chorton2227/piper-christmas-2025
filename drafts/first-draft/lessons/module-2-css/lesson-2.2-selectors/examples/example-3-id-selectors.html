What Are ID Selectors?

ID selectors target ONE specific, unique element on the page. Each ID can only be used once per page. Notice how this entire main content area has a specific ID.

IDs are like social security numbers - each one is unique and can only belong to one person (or in this case, one element).

Why Use IDs?

IDs are useful for major, unique sections of your page:

  • Header area
  • Navigation menu
  • Main content section
  • Sidebar
  • Footer

Each of these major sections typically appears only once on a page, making them perfect candidates for IDs.

IDs vs Classes

Use IDs when: The element is unique (appears only once)

Use Classes when: You want to reuse the style on multiple elements

ID Example:

id="main-header"
#main-header { ... }

Only ONE element can have this ID

Class Example:

class="comparison-box"
.comparison-box { ... }

MANY elements can share this class (like these two boxes!)

Quick Tips

  • IDs must be unique on the page
  • Use # in CSS for IDs
  • Use . in CSS for classes
  • Classes are more common than IDs
