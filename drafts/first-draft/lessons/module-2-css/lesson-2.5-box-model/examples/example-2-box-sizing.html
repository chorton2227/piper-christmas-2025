Box-Sizing: content-box vs border-box

This property changes how width and height are calculated!

What is box-sizing?

The box-sizing property determines whether padding and border are included in the element's width and height. It makes sizing much more predictable!

Side-by-Side Comparison

Both boxes have the same CSS, except for box-sizing. Watch what happens!

content-box (default)

Width: 300px

Padding: 30px

Border: 5px

Total: 370px!

Width applies to content only. Padding and border are added on top.

box-sizing: content-box;

border-box (recommended!)

Width: 300px

Padding: 30px

Border: 5px

Total: 300px!

Width includes content, padding, AND border. Much more intuitive!

box-sizing: border-box;

The Math Behind It

content-box Calculation

Content width: 300px

+ Left padding: 30px

+ Right padding: 30px

+ Left border: 5px

+ Right border: 5px

Total width: 370px

Real-World Examples

Three Equal Columns

With border-box, creating equal-width columns is easy!

Column 1

Width: 33.33%

Padding: 20px

Border: 2px

Still 33.33%!

Column 2

Width: 33.33%

Padding: 20px

Border: 2px

Still 33.33%!

Column 3

Width: 33.33%

Padding: 20px

Border: 2px

Still 33.33%!

Without border-box, these columns would break and wrap because padding/border would push them over 100%!

‚≠ê Best Practice: Use border-box Everywhere!

Add this to the top of your CSS:

* {
    box-sizing: border-box;
}

This applies border-box to ALL elements, making sizing predictable and intuitive.

Why border-box is Better:
