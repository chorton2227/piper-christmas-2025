1. Static Positioning (Default)

Normal document flow - elements appear in order

Box 2 (static)
Box 3 (static)

2. Relative Positioning

Moved relative to its original position - keeps space in layout

Box 1 (static)
Box 2 (relative: moved right & down)
Box 3 (static)
.box { position: relative; top: 20px; /* Move down 20px */ left: 30px; /* Move right 30px */ }

‚ú® Notice Box 2's original space is still reserved (gap remains)

3. Absolute Positioning

Positioned relative to nearest positioned ancestor

Parent (position: relative)

Normal Box 1
Absolute (top-right corner)
Absolute (centered)
Normal Box 2
.parent { position: relative; /* Creates positioning context */ } .absolute-corner { position: absolute; top: 10px; right: 10px; } .absolute-center { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); /* Center trick! */ }

üéØ Absolute elements are removed from normal flow
