# Exercise 1: Color Names Practice

## üéØ Goal
Practice using CSS color names to style different elements.

## üìù Instructions

1. **Open your student workspace folder** and create a new file called `color-names.html`

2. **Create an HTML page** with the following content:
   - A header with a title "My Favorite Colors"
   - Three paragraphs, each describing a different color you like
   - A list of at least 5 colors

3. **Style your page** using CSS color names:
   - Make the page background `lightcyan` or `lavender`
   - Make the header background a color of your choice
   - Make the header text white
   - Give each paragraph a different text color (red, blue, green, purple, etc.)
   - Make the list items different colors - get creative!

4. **Add some extra styling** (optional but fun!):
   - Add padding to your header
   - Make text bigger or smaller
   - Try different background colors on your paragraphs

## üí° Hints

- Use the `<style>` tag in your HTML `<head>` for your CSS
- Remember the format: `color: red;` or `background-color: lightblue;`
- You can use any of the 140+ color names HTML supports!
- Common colors: red, blue, green, yellow, orange, purple, pink, brown, black, white
- Light colors: lightblue, lightgreen, lightpink, lightcoral
- Dark colors: darkblue, darkgreen, darkred

## ‚úÖ Success Criteria

- [ ] Page has a colorful header
- [ ] Three or more elements use different colors
- [ ] All colors are using CSS color names (not hex codes)
- [ ] Text is readable (good contrast with backgrounds)
- [ ] Page looks colorful and fun!
