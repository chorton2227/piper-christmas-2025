# Exercise 3: Color Scheme Website

## üéØ Goal
Create a complete webpage using a professional color scheme.

## üìù Instructions

1. **Choose a color scheme** (pick one):
   - **Monochromatic**: Different shades of one color (like all blues)
   - **Complementary**: Two opposite colors (like blue and orange)
   - **Analogous**: Colors next to each other (like blue, purple, pink)
   - **Triadic**: Three evenly-spaced colors (like red, yellow, blue)

2. **Create a file** called `my-website.html` with these sections:
   - **Header**: Site title and navigation menu
   - **Hero section**: Large welcome section with a heading and description
   - **Three content cards**: Each with a heading, paragraph, and button
   - **Footer**: Copyright and contact info

3. **Apply your color scheme** using the 60-30-10 rule:
   - **60%** - Main color for backgrounds
   - **30%** - Secondary color for sections/cards
   - **10%** - Accent color for buttons and highlights

4. **Make it look professional**:
   - Use consistent spacing (padding and margins)
   - Make sure text is readable with good contrast
   - Style buttons to look clickable
   - Round corners on cards (`border-radius`)

## üé® Challenge Ideas

1. **Add hover effects**: Make buttons change color when you hover
2. **Use transparency**: Try RGBA for subtle overlays
3. **Create a dark mode version**: Dark background with light text
4. **Add shadows**: Use `box-shadow` with subtle colors from your scheme
