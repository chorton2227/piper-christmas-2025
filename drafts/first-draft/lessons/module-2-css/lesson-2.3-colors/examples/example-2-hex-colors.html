What are Hex Codes?

Hex codes start with # followed by 6 characters (0-9, A-F)

Format: #RRGGBB where RR=Red, GG=Green, BB=Blue

Hex Shorthand

When pairs match, you can use 3 characters instead of 6:

#FFF = #FFFFFF (White)
#000 = #000000 (Black)
#F00 = #FF0000 (Red)
#0F0 = #00FF00 (Green)
#00F = #0000FF (Blue)
