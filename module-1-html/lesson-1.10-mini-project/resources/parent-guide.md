# Parent Guide: Mini-Project

## \ud83d\udcda Project Overview

Students create a complete 3+ page website using all HTML skills learned in Module 1. This is their capstone project!

## \ud83d\udca1 Supporting Your Child

### Before Starting
- Help them choose a topic they're excited about
- Encourage planning before coding
- Review requirements together

### During the Project
- Check in on progress regularly
- Encourage breaking it into chunks
- Remind them to test frequently
- Celebrate small wins!

### If They Get Stuck
- Review previous lessons together
- Check the requirements checklist
- Test one thing at a time
- Take breaks when frustrated

## \ud83c\udfaf What Success Looks Like

- 3+ working HTML pages
- All pages linked together
- All required elements present
- Content is complete and makes sense
- Pride in their creation!

## \u2705 Assessment Points

- Structure: Navigation, semantic HTML
- Content: All required elements used
- Quality: Working links, valid HTML
- Creativity: Personal and engaging

**This is their chance to shine - encourage creativity while meeting requirements!** \ud83c\udf1f
