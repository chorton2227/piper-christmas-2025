## \ud83d\udccb Project Requirements

### \ud83c\udfdb\ufe0f Structure (Required)
- [ ] **At least 3 HTML pages**
- [ ] **Navigation menu** on every page
- [ ] **Semantic HTML** (header, nav, main, footer)
- [ ] **Consistent layout** across pages

### \ud83d\udcdd Content (Required)
- [ ] **Heading hierarchy** (h1, h2, h3)
- [ ] **5+ paragraphs** per page
- [ ] **2+ lists** (ul/ol)
- [ ] **5+ links** (internal + external)
- [ ] **5+ images** with alt text
- [ ] **1+ table** with proper structure
- [ ] **1+ form** with inputs
- [ ] **Text formatting** (strong, em, mark)

### \u2728 Quality (Required)
- [ ] Proper DOCTYPE and HTML structure
- [ ] All links work
- [ ] All images display with alt text
- [ ] Well-organized content
- [ ] No broken HTML
- [ ] Cohesive, related pages

## \ud83d\udca1 Tips for Success

### Content
- Write naturally - be yourself!
- Use real content, not placeholder text
- Organize with headings
- Make it interesting

### HTML
- Start with structure
- Test frequently in browser
- Use semantic elements
- Comment your code
- Proper indentation

### Images
- Name files: `my-image.jpg` (no spaces!)
- Always add alt text
- Keep file sizes reasonable
- Organize in images/ folder

### Time Management
- Work in chunks
- Build page by page
- Save often (Ctrl+S)
- Take breaks

---

## \u2705 HTML Elements Checklist

### Must Use
- `<!DOCTYPE html>`
- `<html>`, `<head>`, `<body>`
- `<header>`, `<nav>`, `<main>`, `<footer>`
- `<h1>` through `<h3>`
- `<p>`, `<strong>`, `<em>`
- `<ul>/<ol>` with `<li>`
- `<a>` for links
- `<img>` for images
- `<table>`, `<tr>`, `<td>`, `<th>`
- `<form>`, `<input>`, `<label>`, `<button>`
