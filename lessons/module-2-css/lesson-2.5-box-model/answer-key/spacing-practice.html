Understanding Spacing

Padding and Margin in Action!

Padding Creates Inner Space

Notice how this text doesn't touch the edges of the box? That's because we added 25px of padding on all sides. The padding creates breathing room inside the box, making the content more comfortable to read.

Margin Creates Outer Space

The boxes have space between them because we added a 20px bottom margin to each box. Margin pushes elements away from each other, creating clear separation. This makes the page easier to scan and understand.

Borders Show the Box

The 1px solid border helps you see exactly where the padding ends and the margin begins. The border sits between the padding (inside) and the margin (outside). It's a great way to visualize the box model!
