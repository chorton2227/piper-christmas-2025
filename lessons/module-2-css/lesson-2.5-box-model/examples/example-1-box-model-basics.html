The CSS Box Model

Every element is a box with content, padding, border, and margin!

What is the Box Model?

The box model is how CSS calculates the space an element takes up. Understanding it is essential for layout and spacing!

Visual Box Model

Here's a box with visible layers so you can see each part:

CONTENT

This is the actual content - text, images, etc.

Content - The actual stuff
Padding - Space inside (green)
Border - Edge line (blue)
Margin - Space outside (orange)

Padding Examples

Padding adds space INSIDE an element, between content and border.

No Padding

padding: 0;

Medium Padding

Comfortable spacing

padding: 20px;
Large Padding

Lots of space!

padding: 40px;

Margin Examples

Margin adds space OUTSIDE an element, pushing other elements away.

No Margin

Medium Margin (20px)

All Together Now!

Complete Box

Width & Height
