## ğŸ”‘ Quick Reference

### Selector Syntax
```css
/* Element - NO symbol */
p { color: blue; }

/* Class - DOT (.) */
.highlight { background: yellow; }

/* ID - HASH (#) */
#header { background: navy; }
```

### In HTML
```html
<!-- Class - no dot! -->
<p class="highlight">Text</p>

<!-- ID - no hash! -->
<header id="main-header">Header</header>

<!-- Multiple classes - space separated -->
<div class="card featured">Content</div>
```

## ğŸ’¡ Tips

**Remember the Symbols:**
- HTML: `class="name"` and `id="name"` (no symbols!)
- CSS: `.name` for classes, `#name` for IDs

**Common Mistakes to Avoid:**
- âŒ Using dot in HTML: `class=".card"`
- âŒ Forgetting dot in CSS: `card { }`
- âœ… HTML: `class="card"` CSS: `.card { }`

**Multiple Classes:**
- Separate with SPACES: `class="btn btn-primary"`
- NOT dots: `class="btn.btn-primary"`

## ğŸ¨ When to Use Which?

| Want to... | Use This |
|-----------|----------|
| Style ALL paragraphs | Element `p { }` |
| Create reusable button | Class `.btn { }` |
| Style the header once | ID `#header { }` |
| Add multiple styles | Multiple classes |
