üîç Sources/Debugger Tab

Step through your code line by line!

Setting Breakpoints:

  1. Open Sources/Debugger tab
  2. Find your JavaScript file in the file tree
  3. Click on a line number to set a breakpoint (blue highlight)
  4. Run your code - it will pause at the breakpoint!

Debugging Controls:

  • Watch: Add specific variables to monitor
  • Breakpoints: List of all breakpoints you've set

üé® Elements Tab

Inspect and modify HTML and CSS in real-time!

What You Can Do:

1. Inspect Elements:
  • Right-click any element ‚Üí "Inspect"
  • See the HTML structure
  • View applied CSS styles
2. Edit HTML:
  • Double-click text to edit
  • Right-click element ‚Üí "Edit as HTML"
  • Changes are temporary (refresh to reset)
3. Modify CSS:
  • Click any CSS property to edit
  • Add new styles
  • Toggle checkboxes to enable/disable styles

üåê Network Tab

See all files your page loads!

What You Can See:

  • All files loaded (HTML, CSS, JS, images)
  • How long each file took to load
  • File sizes
  • HTTP status codes (200 = success, 404 = not found, etc.)
  • API requests and responses

How to Use:

  1. Open Network tab
  2. Refresh the page (F5)
  3. Watch files load in real-time
  4. Click any file to see details
üí° Debugging Tip: If something isn't loading, check Network tab for 404 errors (file not found) or 500 errors (server error).
