Using CSS selectors to find elements

What is querySelector?

The querySelector() method finds the FIRST element that matches a CSS selector. It's more flexible than getElementById() because you can use any CSS selector pattern.

Syntax:
let element = document.querySelector('css-selector');

Important: Returns the FIRST matching element, or null if none found.
