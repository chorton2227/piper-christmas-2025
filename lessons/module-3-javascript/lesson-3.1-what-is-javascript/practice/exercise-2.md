## üìã Instructions

### Step 3: Create the JavaScript Quiz
In your script tag, create a math quiz with these sections:

#### Section 1: Easy Questions
Print 5 easy math problems and their answers:
- Addition (e.g., 5 + 3)
- Subtraction (e.g., 10 - 4)
- Multiplication (e.g., 6 * 7)
- Division (e.g., 20 / 4)
- One more of your choice

#### Section 2: Medium Questions
Print 3 medium difficulty problems:
- Two-step calculations (e.g., 5 + 3 * 2)
- Problems with parentheses (e.g., (10 + 5) * 2)
- Using modulo (e.g., 17 % 5)

#### Section 3: Word Problems
Solve 2 word problems using JavaScript:
- Example: "If I have 50 dollars and buy 3 items at $12 each, how much money is left?"
- Example: "A game gives you 25 points per level. How many points after 8 levels?"
