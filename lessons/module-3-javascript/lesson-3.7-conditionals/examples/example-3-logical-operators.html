Logical Operators Reference

Logical operators combine multiple conditions and return true or false.

Operator Name Example Description
&& AND true && true ‚Üí true Returns true only if BOTH conditions are true
|| OR true || false ‚Üí true Returns true if AT LEAST ONE condition is true
! NOT !true ‚Üí false Flips the value (true becomes false, false becomes true)

Demo 1: AND Operator (&&)

Both conditions must be true for the result to be true.

if (condition1 && condition2) {
  // Runs only if BOTH are true
}

Truth Table for AND

A B A && B
true true true
true false false
false true false
false false false

Demo 2: OR Operator (||)

At least one condition must be true for the result to be true.

if (condition1 || condition2) {
  // Runs if AT LEAST ONE is true
}

Truth Table for OR

A B A || B
true true true
true false true
false true true
false false false

Demo 3: NOT Operator (!)

Flips the boolean value - true becomes false, false becomes true.

if (!condition) {
  // Runs if condition is FALSE
}

Truth Table for NOT

A !A
true false
false true
