## Requirements

1. Create a complete HTML document
2. Add a heading: "My Weekly Schedule"
3. Create a table with:
   - **First column**: Time periods (Morning, Afternoon, Evening, etc.)
   - **Other columns**: Days of the week (Monday through Sunday)

4. Fill in activities for each time and day
5. Use `<th>` for both the header row AND the first column (time periods)
6. Include at least **3 time periods** and all **7 days**

## Example Structure

```html
<table border="1">
  <tr>
    <th>Time</th>
    <th>Monday</th>
    <th>Tuesday</th>
    <!-- Add more days -->
  </tr>
  <tr>
    <th>Morning</th>
    <td>School</td>
    <td>School</td>
    <!-- Add more activities -->
  </tr>
  <!-- Add more time periods -->
</table>
```
