The Beginner's Guide to HTML

A comprehensive guide for anyone starting their web development journey!

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses tags to define the structure and content of a webpage.

Think of HTML as the skeleton of a website:

  • HTML - The structure (bones)
  • CSS - The appearance (skin and clothes)
  • JavaScript - The behavior (muscles and movement)

Why Learn HTML?

Learning HTML opens up many possibilities:

  • Build your own websites
  • Create online portfolios
  • Understand how the web works
  • Foundation for other web technologies
  • Job opportunities in web development

Essential HTML Tags

Here are the most important tags every beginner should know:

Structure Tags

  • <html> - Root element of the page
  • <head> - Contains metadata and title
  • <body> - Contains visible content
  • <title> - Page title (shows in browser tab)

Content Tags

  • <h1> to <h6> - Headings
    • Use only one h1 per page!
    • Create proper hierarchy
    • Don't skip levels
  • <p> - Paragraphs
  • <ul> and <ol> - Lists
    • Unordered lists (bullets)
    • Ordered lists (numbers)
    • Always use <li> for items
  • <a> - Links to other pages
  • <img> - Images

Formatting Tags

  • <strong> - Bold/important text
  • <em> - Italic/emphasized text
  • <mark> - Highlighted text
  • <br> - Line break
  • <hr> - Horizontal divider

Step-by-Step: Creating Your First Page

Follow these steps to create your first HTML page:

  1. Open a text editor
    • Use Notepad, VS Code, or any text editor
    • Don't use Word or Google Docs!
  2. Start with the basic structure
    • Type <!DOCTYPE html> at the top
    • Add <html> tags
    • Create <head> and <body> sections
  3. Add a title
    • Put <title> inside the head
    • This shows in the browser tab
  4. Add content to the body
    • Start with an <h1> heading
    • Add paragraphs with <p>
    • Include lists, formatting, etc.
  5. Save the file
    • Save as index.html
    • Make sure it ends with .html!
    • Don't save as .txt
  6. Open in a browser
    • Double-click the file
    • Or drag it into your browser
    • Voilà! You've made a webpage!

Common Mistakes to Avoid

Here are the most common errors beginners make:

  • Forgetting to close tags
    • Every opening tag needs a closing tag
    • Example: <p>Text</p>
    • Exception: Self-closing tags like <br> and <hr>
  • Using multiple h1 tags
    • Only use ONE <h1> per page
    • Use <h2>, <h3>, etc. for other headings
  • Skipping heading levels
    • Don't go from h1 to h4
    • Follow the order: h1 → h2 → h3
  • Not using proper indentation
    • Indentation makes code readable
    • Helps you see structure clearly
    • Makes debugging easier
  • Forgetting the DOCTYPE
    • Always start with <!DOCTYPE html>
    • Tells browser it's HTML5

Tips for Success

Follow these tips to become a great HTML developer:

  1. Practice every day
    • Even 15 minutes makes a difference
    • Build small projects regularly
  2. View source code of websites
    • Right-click → View Source
    • See how professionals structure pages
    • Learn from real examples
  3. Use browser dev tools
    • Press F12 to open developer tools
    • Inspect elements on the page
    • See how HTML, CSS, and JavaScript work together
  4. Don't be afraid to experiment
    • Try new tags and combinations
    • You can't break your computer!
    • Learn from mistakes
  5. Build real projects
    • About Me page
    • Recipe collection
    • Portfolio of your work
    • Fan page for something you love
