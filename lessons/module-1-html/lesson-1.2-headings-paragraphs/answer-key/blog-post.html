Understanding HTML Structure

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses a series of elements to define the structure and content of a webpage.

Each HTML document has a head and a body. The head contains meta-information about the document, while the body contains the content that is displayed in the browser.

Common HTML Elements

Some of the most common HTML elements include headings (h1 to h6), paragraphs (p), links (a), images (img), and lists (ul, ol, li).

Headings are used to create titles and subtitles, while paragraphs are used for blocks of text. Links allow users to navigate between different pages, and images can be embedded to enhance the visual appeal of a webpage.

Creating a Simple Webpage

To create a simple webpage, you can start with the basic structure:

<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
</body>
</html>

This code sets up a basic HTML document with a title, a main heading, and a paragraph.
