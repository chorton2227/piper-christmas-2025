Welcome to my comprehensive web development resources guide! This page contains all the best resources I've found for learning HTML, CSS, JavaScript, and web development in general.

Whether you're a complete beginner or looking to advance your skills, you'll find helpful links organized by category below.

How to Use This Guide:

  1. Use the Table of Contents above to jump to any section

    Great for hands-on learning with instant feedback.
  • Khan Academy - Computer programming and web development
    Excellent video explanations and practice exercises.

  • The Net Ninja - Comprehensive tutorial series

Beginner-Friendly Books

  • Don't Make Me Think by Steve Krug (Web Design)

Need Help Getting Started?

Not sure where to begin? Here's my recommended learning path:

  1. Start with freeCodeCamp HTML/CSS curriculum
  2. Practice on CodePen as you learn
  3. Reference MDN Web Docs when stuck
  4. Join the community on Stack Overflow
  5. Build projects using Frontend Mentor

Remember: Learning to code takes time and practice. Be patient with yourself!
