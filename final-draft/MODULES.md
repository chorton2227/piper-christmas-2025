# Modules Overview (Final Draft)

This index outlines modules, lessons, objectives, prerequisites, artifacts, and assessment.

## Module 1: HTML Foundations
- Objectives: Structure content with semantic HTML; headings, paragraphs, lists, links, images, tables, forms.
- Prerequisites: None.
- Lessons:
  - 1.1 What is HTML? (elements, tags, attributes)
  - 1.2 Headings & Paragraphs
  - 1.3 Lists (ul/ol/dl)
  - 1.4 Links (internal, external, anchors)
  - 1.5 Images (alt, captions)
  - 1.6 Tables (thead/tbody/tfoot)
  - 1.7 Divs & Spans (grouping)
  - 1.8 Forms (inputs, labels, accessibility)
  - 1.9 Semantic HTML (header/main/nav/article/section/footer)
- Artifacts: Simple multi-page site.
- Assessment: Checklist + short reflection.

## Module 2: CSS Essentials
- Objectives: Style layout, typography, colors; responsive design; variables; motion.
- Prerequisites: Module 1.
- Lessons:
  - 2.1 CSS Basics (selectors, specificity)
  - 2.2 Typography & Color (variables)
  - 2.3 Box Model & Layout (flex/grid)
  - 2.4 Responsive Design (media queries)
  - 2.5 Components & Utilities (buttons, cards)
  - 2.6 Transitions & Reduced Motion
  - 2.7 Theming (light/dark)
- Artifacts: Styled site with responsive layout.
- Assessment: Rubric + accessibility audit.

## Module 3: JavaScript Fundamentals
- Objectives: Variables, functions, control flow, arrays/objects, DOM, events, state.
- Prerequisites: Modules 1–2.
- Lessons:
  - 3.1 JS Basics (types, operators)
  - 3.2 Functions & Scope
  - 3.3 Arrays & Objects
  - 3.4 Control Flow & Loops
  - 3.5 DOM Manipulation
  - 3.6 Events & Handlers
  - 3.7 State & Data Flow
- Artifacts: Interactive mini-app.
- Assessment: Unit checks + project demo.

## Module 4: Game Building
- Objectives: Plan, build, and polish a browser game; UX, audio, accessibility.
- Prerequisites: Modules 1–3.
- Lessons:
  - 4.1 Game Loop & State
  - 4.2 Rendering & Animations
  - 4.3 Input Handling
  - 4.4 Assets (audio/images) + Fallbacks
  - 4.5 Accessibility (reduced motion, captions)
  - 4.6 Testing Harness & Debugging
  - 4.7 Final Integration & Reflection (Hold’em demo)
- Artifacts: Playable web game.
- Assessment: Playtest + reflection.

## Module 5: Web Fundamentals Plus
- Objectives: Ship projects confidently — version control, tooling, deployment, collaboration, and portfolios.
- Prerequisites: Modules 1–4.
- Lessons:
  - 5.1 Project Setup & Tooling (editors, linters, formatters)
  - 5.2 Git & GitHub Basics (commits, branches, PRs)
  - 5.3 Publishing & Hosting (GitHub Pages/Netlify/Vercel)
  - 5.4 Assets & Performance (images, caching, minification)
  - 5.5 Accessibility & QA (audit, keyboard, reduced motion)
  - 5.6 Documentation & Portfolios (READMEs, demos, presentation)
- Artifacts: Published project and portfolio page.
- Assessment: Release checklist + peer review.

## Cross-Cutting Practices
- Accessibility, consistency, message speed controls, reusable components.
- Parent guides and cheat sheets per lesson.
