## ğŸ’¡ Tips for Success

### Creating Links
- **External links:** Always use `https://` and `target="_blank"`
- **Internal links:** Just use the filename (e.g., `about.html`)
- **Anchor links:** Add `id` to target, use `#id` in link
- **Email links:** Use `mailto:email@example.com`

### Testing Your Links
- Click every link to make sure it works!
- For anchor links, make sure they jump to the right place
- Check that external links open in new tabs
- Hover over links with tooltips to see if they appear

### Organizing Your Work
- Keep all files for Exercise 2 in this same folder
- Use descriptive filenames
- Add comments to remember what sections do
- Test often - don't wait until the end!

## ğŸ†˜ Getting Stuck?

### Common Issues

**"My link doesn't work!"**
- Check you included `https://` for external links
- Verify filename is spelled correctly for internal links
- Make sure all files are in the same folder
- Check that `</a>` closing tag is there

**"Anchor link doesn't jump!"**
- Make sure `id` matches `href` exactly
- Remember to use `#` before the id in href
- IDs can't have spaces - use hyphens
- Each ID must be unique

**"Email link doesn't work!"**
- Use `mailto:` before the email address
- Check for typos in the email
- Some computers may not have email programs set up (this is normal!)
