# How to View HTML Files (Windows)

Option 1: File Explorer
- Open this folder in File Explorer.
- Double-click the `.html` file; it opens in your default browser.

Option 2: Drag & Drop
- Drag the `.html` file onto your browser window.

Option 3: PowerShell (from this folder)
```powershell
Set-Location path\to\lesson-1.4-links\examples
start .\example-1-external-links.html
```
Replace `path\to\lesson-1.4-links\examples` with the actual relative path on your machine.
