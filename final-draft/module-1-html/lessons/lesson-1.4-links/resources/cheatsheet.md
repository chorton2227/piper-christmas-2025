# Links Cheatsheet

Basic Link
- `<a href="URL">Descriptive Text</a>`

External Link (new tab)
- `<a href="https://example.com" target="_blank" rel="noopener">Text</a>`

Internal Link (relative path)
- `<a href="about.html">About</a>`

Anchor Link (in-page)
- Add `id` to target: `<h2 id="contact">Contact</h2>`
- Link to it: `<a href="#contact">Contact section</a>`

Email & Phone
- Email: `<a href="mailto:hello@example.com">hello@example.com</a>`
- Phone: `<a href="tel:+1234567890">+1 (234) 567-890</a>`

Tips
- Avoid vague text like “click here”.
- Keep link text concise and meaningful.
- Provide a skip link to `#main` on longer pages.
