# Exercise 1: Create Your Lists

## ðŸŽ¯ Goal
Practice creating both unordered and ordered lists on a single page. You'll learn when to use each type and how to organize information effectively!

---

## ðŸ“‹ Instructions

### Step 1: Create Your File
Create a new HTML file called `my-lists.html` in your student-workspace folder.

### Step 2: Set Up HTML Structure
Start with the basic template:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Lists</title>
</head>
<body>
    <!-- Your lists go here! -->
</body>
</html>
```

### Step 3: Add a Main Heading
Add an `<h1>` tag with a title for your page (like "All About Me" or "My Favorite Things").

### Step 4: Create Your First Unordered List
1. Add an `<h2>` heading describing what your list is about
2. Create a `<ul>` tag
3. Add at least 5 `<li>` items
4. Close the `</ul>` tag

### Step 5: Create Your First Ordered List
1. Add another `<h2>` heading
2. Create an `<ol>` tag
3. Add at least 5 `<li>` items where order matters
4. Close the `</ol>` tag

### Step 6: Add More Lists
Create at least one more list of each type (total: 2 unordered, 2 ordered minimum).

## ðŸ†˜ Troubleshooting

**Problem:** My list items don't have bullet points or numbers  
**Solution:** Make sure each `<li>` is inside a `<ul>` or `<ol>` container.

**Problem:** My lists run together  
**Solution:** Check that you've closed each `</ul>` or `</ol>` before starting a new list.

**Problem:** Text shows outside the list  
**Solution:** All content should be inside `<li>` tags when in a list.
