# Lists Cheatsheet

Unordered List
- `<ul>` contains list items
- `<li>` represents an item

Ordered List
- `<ol>` for sequences
- `<li>` for each step

Description List
- `<dl>` wraps the list
- `<dt>` is the term
- `<dd>` is the description

Tips
- Choose list type based on meaning (order vs grouping).
- Keep items concise and parallel.
- Use nesting sparingly.
