Learning to Code: My First Week

By: Piper | November 19, 2025 | Learning Journey | 5 min read

Wow! I can't believe it's already been a week since I started learning web development. This journey has been absolutely amazing and I've learned so much already!

What I've Learned

In just one week, I've mastered the basics of HTML. I started with simple things like headings and paragraphs, but now I can create tables, forms, and organize content with divs and spans!

"Learning to code is like learning a superpower. Every new concept unlocks new possibilities!"

- Me, after building my first website

Key Lessons I've Learned

  • Start Simple: Don't try to build everything at once. Master the basics first!
  • Practice Daily: Even 30 minutes of coding each day makes a huge difference.
  • Don't Fear Errors: Every error is a learning opportunity, not a failure.
  • Have Fun: Choose projects you're excited about. Learning is easier when you're having fun!
  • Ask Questions: There's no such thing as a "dumb question" when you're learning.

My Favorite Moments

One of my proudest moments was when I created my first multi-page website with working navigation. Seeing all the pages connect together was incredibly satisfying!

Another highlight was learning about tables. At first, I thought they'd be complicated, but once I understood the pattern of rows and cells, it became super easy!

Challenges I Faced

Not everything was smooth sailing. I definitely struggled with:

  • Forgetting to close tags (so many </div> tags!)
  • Understanding when to use divs vs spans
  • Keeping my code organized with proper indentation
  • Debugging when something didn't look right

But with patience and practice, I overcame each challenge. That's what makes coding so rewarding!

What's Next?

Next week, I'll be diving into CSS! I can't wait to add colors, layouts, and beautiful styling to all the HTML pages I've created. The thought of making my websites look professional is super exciting!

After CSS, I'll learn JavaScript to make everything interactive. Eventually, I want to build a browser-based game that combines everything I've learned!

If you're thinking about learning to code, just start! Don't wait for the "perfect time" or worry about being "ready." The best way to learn is by doing. Build something simple, make mistakes, and learn from them!
