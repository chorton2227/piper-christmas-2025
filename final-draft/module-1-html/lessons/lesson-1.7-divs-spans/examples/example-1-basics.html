Understanding Divs and Spans

Notice how each div is on its own line, even though they're short!

Example 2: Basic Spans (Inline)

Spans stay on the same line:

This is span one. This is span two. This is span three.

All three spans are on the same line!

Example 3: Spans in Paragraphs

My favorite games are Minecraft, Roblox, and Fortnite. The word awesome describes them all!

Right now the spans look the same, but later with CSS we can make specific words look different!

Example 4: Divs Grouping Content

Section 1: Introduction

This heading and paragraph are grouped inside one div.

Multiple elements can be inside a single div!

Section 2: Details

This is a separate div with its own heading and content.

  • Divs can contain any elements

Example 5: The Difference

With divs (each on new line):

Box 1
Box 2
Box 3

With spans (all on same line):

Box 1 Box 2 Box 3

See the difference? Divs stack vertically, spans flow horizontally!
