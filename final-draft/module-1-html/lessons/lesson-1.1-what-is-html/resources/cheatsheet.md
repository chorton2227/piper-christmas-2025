# HTML Cheatsheet (Lesson 1.1)

Structure
- `<!DOCTYPE html>` — HTML5 doctype
- `<html lang="en">` — root element
- `<head>` — metadata: `<meta charset>`, `<title>`
- `<body>` — visible content

Content
- Headings: `<h1>` to `<h6>`
- Paragraph: `<p>`
- List: `<ul>`, `<ol>`, `<li>`
- Link: `<a href="...">text</a>`
- Image: `<img src="..." alt="...">`

Tips
- Always set a meaningful `<title>`.
- Use semantic elements for clarity and accessibility.
- Always include `alt` on images.
