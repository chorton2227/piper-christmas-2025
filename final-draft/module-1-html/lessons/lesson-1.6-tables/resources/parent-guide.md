# Parent Guide: Lesson 1.6 - Tables

## \ud83d\udcda Lesson Overview
Students learn to create tables for organizing data in rows and columns.

## \ud83d\udca1 Teaching Tips
- Tables are like spreadsheets\n- Each row must have same number of cells\n- Headers help organize information\n\n## \ud83d\udd27 Common Challenges\n**Missing closing tags:** Each `<tr>`, `<td>`, `<th>` needs to close\n**Uneven rows:** All rows need same cell count\n\n## \ud83c\udfaf Assessment\n- [ ] Creates basic tables\n- [ ] Uses headers correctly\n- [ ] Organizes with sections\n\n**Keep encouraging!** \ud83c\udf89
