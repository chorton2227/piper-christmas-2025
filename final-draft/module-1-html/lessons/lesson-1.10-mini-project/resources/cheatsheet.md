# Mini-Project Quick Reference

## \u2705 Requirements At-a-Glance

### Pages (3+)
- index.html
- Content page  
- Contact page

### Elements Must Include
- Semantic HTML (header, nav, main, footer)
- h1, h2, h3 headings
- 5+ paragraphs per page
- 2+ lists (ul/ol)
- 5+ links (internal + external)
- 5+ images with alt text
- 1+ table
- 1+ form
- Text formatting (strong, em, mark)

### Quality Checks
- All links work
- All images display
- Navigation on every page
- No broken HTML
- Well organized

## \ud83d\ude80 Build Order

1. Create file structure
2. Add HTML structure to all pages
3. Add navigation to all pages
4. Fill in content page by page
5. Test everything
6. Polish and perfect!

**Use this as your quick checklist!** \u2705
