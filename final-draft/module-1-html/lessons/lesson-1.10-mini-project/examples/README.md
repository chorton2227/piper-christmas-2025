## How to View the Example

1. Navigate to: `../example-project/`
2. Open `index.html` in your browser
3. Explore all the pages
4. View the source code to see how it's built
5. Use it as inspiration for your own project!

## What You'll See

The example project is **"Ultimate Gaming Hub"** - a website about video games featuring:
- **index.html** - Home page with welcome and highlights
- **reviews.html** - Game reviews with images and tables
- **tips.html** - Gaming tips with lists
- **contact.html** - Contact form and feedback

This shows you what a complete project looks like. Your project will be on your own topic, but should have similar structure and elements!
