# Mini-Project: Build Your Complete Website!

## \ud83c\udfaf Your Assignment

Create a complete 3+ page website about a topic you love, using all HTML skills from Module 1.

## \ud83d\udcdd What to Create

Choose ONE topic:
- \ud83c\udfae Gaming site (reviews, tips, favorites)
- \ud83c\udfa5 Fan site (show/movie/book)
- \ud83c\udfa8 Hobby showcase
- \ud83c\udf55 Recipe collection
- \ud83c\udf0e Travel guide
- \ud83d\udc3e Pet/animal site
- Your own idea!

## \ud83d\udccb Required Pages

### index.html - Home Page
- Welcome heading (h1)
- Introduction paragraphs
- Navigation menu
- Featured content
- Images

### [topic].html - Main Content
- Detailed information
- Lists
- Table with data
- More images
- External links

### contact.html - Contact/Form Page
- Contact form
- Input fields (text, email, textarea)
- Submit button
- Contact info

## \u2705 Requirements Checklist

### Structure
- [ ] 3+ HTML pages
- [ ] Navigation on every page
- [ ] Semantic HTML (header, nav, main, footer)

### Content
- [ ] Heading hierarchy (h1, h2, h3)
- [ ] 5+ paragraphs per page
- [ ] 2+ lists
- [ ] 5+ links
- [ ] 5+ images with alt text
- [ ] 1+ table
- [ ] 1+ form
- [ ] Text formatting (strong, em)

### Quality
- [ ] All links work
- [ ] All images display
- [ ] No broken HTML
- [ ] Well organized

## \ud83d\ude80 Getting Started

1. Choose your topic
2. Plan your 3 pages
3. Create files in student-workspace
4. Build structure first
5. Add content
6. Test everything
7. Polish and perfect!
