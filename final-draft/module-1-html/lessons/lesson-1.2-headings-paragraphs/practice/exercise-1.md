# Exercise 1 — Blog Outline

Create `blog-outline.html`:
- Topic: A hobby or interest of yours.
- Use a single `<h1>` for the page title.
- Add at least two `<h2>` sections.
- Under each `<h2>`, add 1–2 paragraphs.

Checklist:
- One `<h1>` only.
- Headings summarize following content.
- No skipped levels.
