⚖️ Example 2: Comparison Operators

Comparison Operators Reference

Comparison operators compare two values and return true or false.

Operator Name Example Description
=== Strict Equal 5 === 5 → true Checks if values are exactly equal (same type and value)
!== Strict Not Equal 5 !== 3 → true Checks if values are not equal
> Greater Than 10 > 5 → true Checks if left value is greater than right value
< Less Than 3 < 8 → true Checks if left value is less than right value
>= Greater or Equal 5 >= 5 → true Checks if left value is greater than or equal to right value
<= Less or Equal 4 <= 7 → true Checks if left value is less than or equal to right value
⚠️ Important: Always use === and !== (strict comparison) instead of == and != (loose comparison). Strict comparison is more reliable and prevents unexpected bugs!
