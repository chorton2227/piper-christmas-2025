🛠️ Array Methods - Interactive Examples

Demo 1: push() and pop() - Add/Remove from End

push(item)
Adds one or more items to the END of an array. Returns new length.
pop()
Removes the LAST item from an array. Returns the removed item.

Demo 2: unshift() and shift() - Add/Remove from Beginning

unshift(item)
Adds one or more items to the BEGINNING of an array. Returns new length.
shift()
Removes the FIRST item from an array. Returns the removed item.

Demo 3: splice() - Add/Remove at Any Position

splice(start, deleteCount, ...items)

Demo 4: slice() - Copy Part of Array

slice(start, end)
Returns a shallow copy of a portion of an array. Does NOT modify original.

Demo 5: join() - Array to String

join(separator)
Joins all array elements into a string, separated by a specified separator.
Array: ['Hello', 'World', 'From', 'JavaScript']

Demo 6: concat() - Combine Arrays

concat(array1, array2, ...)
Merges two or more arrays into a new array. Does not modify originals.

Demo 7: reverse() and sort() - Reorder Elements

reverse()
Reverses the order of elements in an array. Modifies the original array.
sort()
Sorts elements in an array. Modifies the original array.
