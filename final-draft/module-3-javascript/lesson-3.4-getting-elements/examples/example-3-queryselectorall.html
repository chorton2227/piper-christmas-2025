🎯 querySelectorAll Basics

Getting ALL elements that match a selector

What is querySelectorAll?

The querySelectorAll() method finds ALL elements that match a CSS selector and returns them in a NodeList (similar to an array). Unlike querySelector() which only gets the first match, this gets them all!

Syntax:
let elements = document.querySelectorAll('css-selector');

Returns: A NodeList containing all matching elements (or an empty NodeList if none found).

Important: NodeList is similar to an array - you can access elements with [0], [1], etc., and it has a .length property.

Console Output

Check your browser console (F12) to see all the elements!

// Get ALL enemy cards
let enemies = document.querySelectorAll('.enemy-card');
console.log(enemies.length);
→ 3

// Access individual enemies
console.log(enemies[0]);
→ <div class="enemy-card">Slime...</div>
