Using Hex Codes for Precise Colors

Hex codes give you access to 16.7 MILLION colors! Format: #RRGGBB where each pair goes from 00 to FF.

Understanding Hex Codes

#FF0000

Pure Red

RR=FF (max), GG=00 (none), BB=00 (none)

#00FF00

Pure Green

RR=00, GG=FF (max), BB=00

#0000FF

Pure Blue

RR=00, GG=00, BB=FF (max)

Mixed Colors

#FFFF00

Yellow

Red + Green = Yellow

#00FFFF

Cyan

Green + Blue = Cyan

#FF00FF

Magenta

Red + Blue = Magenta
