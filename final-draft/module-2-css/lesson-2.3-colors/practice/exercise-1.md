### Step 3: Style Using Color Names ONLY
Use only named colors (no hex/RGB this exercise):

**Required Styles:**
```css
body {
    background-color: lightcyan;  /* or lavender, mint cream, etc. */
}

h1 {
    background-color: navy;
    color: white;
    padding: 20px;
}

.para1 { color: darkblue; }
.para2 { color: darkgreen; }
.para3 { color: purple; }
```

Style each paragraph and list item with different colors!
