## üìã Project Requirements

### Required Pages/Sections:

1. **Header/Navigation**
   - Fixed or sticky position
   - Flexbox navigation
   - Hover effects on links
   - Responsive (hamburger menu optional)

2. **Hero Section**
   - Large gradient or image background
   - Centered content
   - Call-to-action button with hover effect
   - Full viewport height

3. **About Section**
   - Box model (padding, margins)
   - Typography (Google Fonts)
   - Background color or subtle gradient

4. **Projects/Portfolio Grid**
   - Flexbox or Grid layout
   - 3-6 project cards
   - Borders, shadows, rounded corners
   - Hover effects (transform, shadow)
   - Responsive (wraps on mobile)

5. **Skills/Services Section**
   - Icon cards or list
   - Color-coded or gradient backgrounds
   - Proper spacing

6. **Contact/CTA Section**
   - Form or contact links
   - Different background (gradient/color)
   - Centered content

7. **Footer**
   - Links, copyright
   - Background color
   - Padding, borders

## üé® CSS Skills to Demonstrate

Must include:
- [ ] **Selectors**: Class, ID, element selectors
- [ ] **Colors**: Hex codes, RGB/RGBA
- [ ] **Typography**: Font families, sizes, weights, Google Fonts
- [ ] **Box Model**: Padding, margin, borders, width/height
- [ ] **Borders**: Styles, colors, border-radius
- [ ] **Backgrounds**: Colors, gradients, images (optional)
- [ ] **Flexbox**: justify-content, align-items, flex-wrap
- [ ] **Positioning**: At least fixed header or sticky nav
- [ ] **Transitions**: Hover effects on buttons and cards
- [ ] **Responsive Design**: Media queries, mobile-first

---

## üéØ Design Requirements

**Color Scheme:**
- Choose 3-5 colors
- Use consistently throughout
- Ensure good contrast

**Typography:**
- 1-2 Google Fonts
- Clear hierarchy (H1, H2, P sizes)
- Readable line-height

**Spacing:**
- Consistent padding/margins
- Sections well-separated
- Content not touching edges

**Visual Polish:**
- Box-shadows for depth
- Rounded corners (border-radius)
- Smooth transitions
- Professional appearance
