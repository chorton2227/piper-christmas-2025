Understanding RGB

RGB stands for Red, Green, Blue - the three colors of light that mix to create all colors!

Format: rgb(red, green, blue) where each value is 0-255

  • rgb(0, 0, 0) = Black (no light)
  • rgb(255, 255, 255) = White (all light)
  • rgb(255, 0, 0) = Red
  • rgb(0, 255, 0) = Green
  • rgb(0, 0, 255) = Blue
