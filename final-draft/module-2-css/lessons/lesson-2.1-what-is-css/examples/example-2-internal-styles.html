Internal CSS Example

All the CSS for this page is in the <style> tag in the <head> section. This keeps our styles organized in one place!

Benefits of Internal CSS

Internal CSS is better than inline styles because:

  • All styles are in one place (the head section)
  • You can see all your styles at once
  • You can apply the same style to multiple elements
  • It's easier to maintain than inline styles
