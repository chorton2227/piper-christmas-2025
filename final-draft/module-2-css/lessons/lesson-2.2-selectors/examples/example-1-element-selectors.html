Element Selectors Demo

This example demonstrates element selectors. All paragraphs will have the same styling because we're targeting the <p> element itself.

What are Element Selectors?

Element selectors target ALL elements of a specific type. Notice how ALL paragraphs on this page have the same font size and color.

Examples of Elements Being Styled

Every paragraph element gets styled exactly the same way. This is useful for setting consistent typography across your entire website.

  • All list items also share the same style
  • They get styled together as <li> elements
  • This creates visual consistency

Smaller Heading
