# Exercise 1: Border Styling Practice

## üéØ Objective
Practice using different border properties including styles, widths, colors, and border-radius.

## üìù Instructions

Create a file called `border-practice.html` in your student workspace and add these elements with the specified border styles:

### Part 1: Border Styles (6 boxes)

Create 6 boxes, each demonstrating a different border style:

1. **Solid Border Box**
   - 3px solid blue border
   - 15px padding
   - Light gray background

2. **Dashed Border Box**
   - 3px dashed red border
   - 15px padding
   - Light yellow background

3. **Dotted Border Box**
   - 3px dotted green border
   - 15px padding
   - Light blue background

4. **Double Border Box**
   - 5px double purple border
   - 15px padding
   - Light pink background

5. **Groove Border Box**
   - 6px groove orange border
   - 15px padding
   - Light peach background

6. **Ridge Border Box**
   - 6px ridge teal border
   - 15px padding
   - Light cyan background

### Part 2: Individual Borders (3 boxes)

Create 3 boxes with borders on specific sides only:

1. **Top Border Only**
   - 4px solid blue border on top only
   - 20px padding
   - White background

2. **Left Border Accent**
   - 6px solid green border on left only
   - 20px padding
   - Light gray background

3. **Multi-Color Border**
   - Top: 3px solid red
   - Right: 3px solid orange
   - Bottom: 3px solid green
   - Left: 3px solid blue
   - 20px padding

### Part 3: Border Radius (4 boxes)

Create 4 boxes demonstrating border-radius:

1. **Slightly Rounded**
   - 10px border-radius
   - 2px solid gray border
   - 20px padding

2. **Very Rounded**
   - 30px border-radius
   - 2px solid blue border
   - 20px padding

3. **Circle**
   - 50% border-radius
   - 2px solid purple border
   - Make it square (same width and height: 150px)

4. **Pill Shape Button**
   - 50px border-radius
   - No border
   - Blue background
   - White text
   - 15px vertical padding, 40px horizontal padding

## üí° Tips

- Use a container with `max-width: 800px` and center it with `margin: 0 auto`
- Add `margin-bottom: 20px` to separate boxes
- Use headings (`<h3>`) inside each box to label them
- Use `box-sizing: border-box` for easier sizing
- Display boxes in a grid or stack them vertically

## ‚úÖ Success Criteria

Your page should have:
- All 13 elements with correct border properties
- Clear labels for each border style
- Consistent spacing and padding
- Clean, organized layout
- Descriptive headings
