TEXT PROPERTIES

Master the art of text styling!

Font Size

Tiny text (12px) - too small for body text!

Small text (14px) - still a bit small

Normal text (16px) - perfect for reading!

Large text (20px) - great for emphasis

Huge text (32px) - attention-grabbing!

font-size: 16px;

Font Weight

Thin weight (100) - very light

Light weight (300) - elegant

Normal weight (400) - standard text

Semibold weight (600) - slightly bold

Bold weight (700) - strong emphasis!

Black weight (900) - maximum impact!

font-weight: bold; /* or 700 */

Text Align

Left aligned (default) - This text is aligned to the left side.

Center aligned - This text is centered in the middle.

Right aligned - This text is aligned to the right side.

Justified - This text is spread out to fill the entire width, creating straight edges on both sides. This is often used in newspapers and magazines to create clean columns of text.

text-align: center;

Line Height (Spacing Between Lines)

Cramped (line-height: 1.0)

This text has very tight line spacing. It's hard to read because the lines are too close together. Your eyes have to work harder to follow from one line to the next. Not recommended for body text!

Normal (line-height: 1.5)

This text has comfortable line spacing. It's easy to read because there's enough space between lines. This is the sweet spot for most body text. Your eyes can easily move from line to line.

Spacious (line-height: 2.0)

This text has very generous line spacing. There's lots of room between lines, which can be nice for emphasis or special sections, but might be too much for long paragraphs.

line-height: 1.6;

Letter Spacing

Tight letters (letter-spacing: -1px)

Normal letters (letter-spacing: 0)

Spaced letters (letter-spacing: 2px)

W I D E L E T T E R S (letter-spacing: 8px)

letter-spacing: 2px;

Text Decoration

Underlined text - draws attention

Overlined text - unique style

Strikethrough text - shows deletion

No decoration - clean and simple

Styled link without underline text-decoration: underline;

Text Transform

uppercase - makes everything capitals!

LOWERCASE - makes everything lowercase!

capitalize - first letter of each word

None - leaves text as-is: Hello World

text-transform: uppercase;

Font Style

Normal style - regular upright text

Italic style - slanted for emphasis

Oblique style - artificially slanted

font-style: italic;
