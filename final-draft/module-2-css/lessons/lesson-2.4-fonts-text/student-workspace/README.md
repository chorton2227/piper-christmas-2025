## Font Selection Guide

**For Headings:**
- Bold, attention-grabbing
- Can be decorative
- Larger size (32-48px)

**For Body Text:**
- Easy to read
- Not too decorative
- Good size (16-18px)
- Comfortable line spacing (1.5-1.8)

## Quick Reference

```css
/* Make text bigger */
font-size: 24px;

/* Make text bold */
font-weight: bold;

/* Center text */
text-align: center;

/* Add space between lines */
line-height: 1.6;

/* Change font */
font-family: Arial, sans-serif;
```

## Google Fonts Quick Start

1. Go to **fonts.google.com**
2. Find a font you like
3. Click "Get Font" then "Get embed code"
4. Copy the `<link>` into your HTML `<head>`
5. Use it in CSS: `font-family: 'Font Name', sans-serif;`

## Typography Checklist

Before you finish each project, ask yourself:

- [ ] Is my text easy to read?
- [ ] Are my headings clearly different from body text?
- [ ] Did I use appropriate font sizes?
- [ ] Is there enough space between lines?
- [ ] Do my fonts work well together?
- [ ] Would I want to read a whole article in these fonts?
