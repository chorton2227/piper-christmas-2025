## ğŸ’¡ Tips

- **Element selectors** style ALL elements of that type (`h1`, `p`, `div`)
- **Class selectors** style specific groups (`.highlight`, `.btn`, `.card`)
- **ID selectors** style one unique element (`#header`, `#main`, `#footer`)
- In HTML: `class="name"` (no dot!)
- In CSS: `.name { }` (with dot!)
- In HTML: `id="name"` (no hash!)
- In CSS: `#name { }` (with hash!)
