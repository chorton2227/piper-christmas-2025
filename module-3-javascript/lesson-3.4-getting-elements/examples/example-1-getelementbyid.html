📦 getElementById Basics

Learning to find HTML elements by their unique ID

What You'll Learn

The getElementById() method is the most straightforward way to find a specific element on your page. Every element can have a unique id attribute, and this method finds that exact element.

Syntax:
let element = document.getElementById('element-id');

Important: Don't include the # symbol - that's only for CSS!

Console Output

Check your browser console (F12) to see the actual elements! Here's what the code does:

// Example 1: Getting a heading
let gameTitle = document.getElementById('game-title');
console.log(gameTitle);
→ <h3 id="game-title">Epic Adventure RPG</h3>

// Example 2: Getting stat elements
let playerHealth = document.getElementById('player-health');
→ <div class="stat-box" id="player-health">...</div>

// Example 3: Element doesn't exist
let missing = document.getElementById('does-not-exist');
→ null
