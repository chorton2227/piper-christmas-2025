### Step 4: Write JavaScript Code

1. A greeting message (e.g., "Hello! My name is...")
2. Your age (use math if you want! Like `2025 - 2015`)
3. Your favorite color
4. Your favorite number
5. A math fact using your favorite number (e.g., "My favorite number doubled is...")
6. Three things you like to do
