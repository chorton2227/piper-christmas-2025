## 📚 Quick Reference

### Common Code Snippets

**Display text:**
```javascript
document.getElementById('elementId').textContent = 'text';
```

**Create button:**
```javascript
const button = document.createElement('button');
button.textContent = 'Click Me';
button.onclick = functionName;
```

**Random number:**
```javascript
Math.floor(Math.random() * max) + min;
```

**Check if exists:**
```javascript
if (variable) {
    // it exists
}
```

**Loop through object:**
```javascript
for (let key in object) {
    const value = object[key];
}
```

### Debugging Checklist

When something doesn't work:
1. [ ] Check browser console (F12)
2. [ ] Read error message carefully
3. [ ] Add console.log statements
4. [ ] Check spelling and capitalization
5. [ ] Verify function is called
6. [ ] Confirm element IDs match
7. [ ] Check if variable is defined
8. [ ] Test with simple values
9. [ ] Compare to working example
10. [ ] Take a break and return

For when things get tough!

- Every expert was once a beginner
- Bugs are proof you're trying
- Each error teaches something new
- The journey matters more than perfection
- You're building something REAL
- Take breaks when needed
- Ask for help - it's not cheating!
- Celebrate small victories

### Encouraging Quotes:
> "The only way to learn a new programming language is by writing programs in it." - Dennis Ritchie

> "Code is like humor. When you have to explain it, it's bad." - Cory House

> "First, solve the problem. Then, write the code." - John Johnson

### When Stuck:
1. Take a 5-minute break
2. Explain the problem out loud
3. Check one more time
4. Look at the examples
5. Ask for help
6. Try tomorrow
7. Remember why you started
