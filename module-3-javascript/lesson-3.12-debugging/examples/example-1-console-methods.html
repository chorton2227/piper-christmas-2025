🔍 Console Methods Demo

📢 Important: Open your browser's Developer Tools (Press F12) and go to the Console tab to see all the output!

Demo 1: console.log() - Basic Logging

The most common method - prints messages to the console.

console.log("Hello, World!"); console.log("Value:", 42); console.log("Multiple:", 1, 2, 3);

Demo 2: console.error() & console.warn()

Show errors in red and warnings in yellow.

console.error("This is an error!"); console.warn("This is a warning!");

Demo 3: console.table() - Display Data Tables

Perfect for viewing arrays of objects!

let students = [ { name: "Alice", grade: 95 }, { name: "Bob", grade: 87 } ]; console.table(students);

Demo 4: console.time() - Measure Performance

See how long code takes to run.

console.time("myTimer"); // ... code to measure ... console.timeEnd("myTimer");

Demo 5: console.count() - Count Function Calls

Track how many times something happens.

function myFunction() { console.count("Function called"); }

Demo 6: console.group() - Organize Output

Group related messages together.

console.group("User Info"); console.log("Name: Alice"); console.log("Age: 25"); console.groupEnd();

Demo 7: console.assert() - Test Conditions

Only shows message if condition is FALSE.

let age = 15; console.assert(age >= 18, "Must be 18+");

Demo 8: console.clear() - Clear Console

Clean slate! Removes all console output.
