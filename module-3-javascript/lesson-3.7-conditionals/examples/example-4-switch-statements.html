🔀 Example 4: Switch Statements

What is a Switch Statement?

A switch statement is like a more organized way to check for multiple specific values. Instead of writing many if/else if statements, you can use a switch!

switch (variable) {
  case value1:
    // Code if variable === value1
    break;
  case value2:
    // Code if variable === value2
    break;
  default:
    // Code if no cases match
}
⚠️ Important: Always include break; at the end of each case! Without it, the code will "fall through" to the next case.

Using If/Else

if (fruit === 'apple') {
  color = 'red';
} else if (fruit === 'banana') {
  color = 'yellow';
} else if (fruit === 'grape') {
  color = 'purple';
} else {
  color = 'unknown';
}

Using Switch

switch (fruit) {
  case 'apple':
    color = 'red';
    break;
  case 'banana':
    color = 'yellow';
    break;
  case 'grape':
    color = 'purple';
    break;
  default:
    color = 'unknown';
}

Switch statements are cleaner when checking one variable against many specific values!
